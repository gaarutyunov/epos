Epos

A Go-based packaging, distribution, and registry-proxy system for AI-agent Skills — Helm's authoring-and-distribution surface, reimagined for Skills as versioned, templatable OCI artifacts.

  • OCI-native packaging with Epos media types (via ORAS)
  • Digest-pinned, rollback-able local installs and mountable ConfigMaps
  • Kustomize-style overlays and unified OCI/git dependencies
  • Transparent credential pass-through proxy with download stats
  • cosign/Sigstore verify-when-present signing

See the repository and SPEC.md for the full specification.
